# 📊 ShopStream Product Analytics Dashboard

## Overview

This project was built to replicate how a **Product Analyst** works in a real business environment.

Rather than focusing only on dashboard creation, I followed a complete product analytics workflow—from raw transactional data to actionable business insights. Using SQL and Power BI, I cleaned and transformed raw data, calculated key product metrics, analyzed customer behavior, and developed interactive dashboards to support data-driven decision-making.

The project covers essential Product Analytics concepts including Funnel Analysis, Cohort Analysis, RFM Segmentation, Churn Analysis, KPI Tracking, Customer Retention, Revenue Analysis, and Customer Segmentation.
